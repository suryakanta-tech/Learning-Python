Packages : 
=============
Packages is a group of python files. In each python file in a package there may be one or more functions written in it.


How to install a package ?
----------------------------
PIP : Python Package Installer

example : 
----------
PIP install packageName (to install a package)

PIP list (To check a package is installed or not)

eg. NumPy, Panda, Matplotlib etc.


Module : 
----------

e.g.

pyplot, random, os etc
